# Excel Skills for Business: Intermediate I

This repository contains my practice work from the **Excel Skills for Business: Intermediate I** course (part of the Excel Skills for Business Specialization by Macquarie University on Coursera).

---

## Purpose

This repository is created to:

- Strengthen intermediate-level Excel skills through structured hands-on practice  
- Document weekly learning progress  
- Maintain consistent GitHub activity  
- Build stronger spreadsheet management and analytical skills  

This repository serves as a structured learning archive rather than a portfolio project.
